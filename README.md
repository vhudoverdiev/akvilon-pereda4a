# Отдел передач — Аквилон

Готовое учебное веб-приложение на **Python + Flask + SQLite**.

## Что уже есть

- авторизация в админ-панель по логину и паролю;
- объект **Квартал 100-7**;
- раздел **Запуск объекта**;
- форма добавления записи;
- таблица с полями:
  - номер квартиры,
  - стекла,
  - пол,
  - потолок,
  - дополнительные;
- удаление записи;
- стильный темный интерфейс в холодных аква-тонах.

## Как запустить

### 1. Перейдите в папку проекта

```bash
cd aquilon_app
```

### 2. Создайте виртуальное окружение

**Windows:**
```bash
python -m venv venv
venv\Scripts\activate
```

**macOS / Linux:**
```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Запустите приложение

```bash
python app.py
```

### 5. Откройте браузер

```text
http://127.0.0.1:5000
```

## Как это работает

### `app.py`
Это главный Python-файл.
Он:
- запускает Flask;
- создает базу SQLite;
- проверяет логин и пароль;
- показывает страницы;
- добавляет записи в таблицу;
- удаляет записи.

### `templates/`
Здесь лежат HTML-шаблоны.
Flask подставляет в них данные из Python.

### `static/style.css`
Это стили сайта:
- цвета,
- кнопки,
- формы,
- таблица,
- фон,
- адаптивность под телефон.

## Где поменять логин и пароль

Откройте файл `app.py` и найдите строки:

```python
app.config["ADMIN_LOGIN"] = os.environ.get("ADMIN_LOGIN", "admin")
app.config["ADMIN_PASSWORD"] = os.environ.get("ADMIN_PASSWORD", "aquilon123")
```

Можно изменить значения прямо там.
